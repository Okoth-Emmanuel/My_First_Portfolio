About This Project

This project demonstrates basic, valid HTML5 structure. It is part of my learning path to build simple web pages using only HTML, CSS, and JavaScript.

The goal is to practice semantic markup, headings, paragraphs, and links. All files are kept minimal and are validated with the W3C HTML validator.

What I Learned

I learned how to structure content using headings (<h1>, <h2>, <h3>) and paragraph tags (<p>).

I also practiced making images clickable by wrapping an <img> inside an <a> element, and the importance of providing descriptive alt text for accessibility.
